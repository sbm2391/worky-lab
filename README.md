# Worky-lab

Technologies used: MongoDB, Express, Vuejs, Node.js, Vuetify.

## Instructions

- Clone this repo
- Install the dependencies in the local node_modules from your terminal
```
$ cd worky-lab
$ cd server
$ npm install
```
```
$ cd worky-lab
$ cd client
$ npm install
```
- Run the seed file
```
$ cd server
$ cd bin
$ node seeds.js
```
- inicialice the project
```
$ cd worky-lab
$ cd server
$ npm start
```
```
$ cd worky-lab
$ cd client
$ npm start
```
